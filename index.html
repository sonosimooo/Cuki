Autonomous Navigation

The heart of Cuki M-1's intelligence lies in its autonomous navigation system. Using the ultrasonic sensor, the robot continuously scans its environment and makes real-time decisions to avoid obstacles.

// Main autonomous navigation function
void autonomousNavigation() {
  int distance = getDistance();  // Read ultrasonic sensor
  
  if (distance > 20) {          // Clear path ahead
    moveForward();
  } else {
    stopMotors();
    int rightDistance = lookRight();
    int leftDistance = lookLeft();
    
    if (rightDistance > leftDistance) {
      turnRight();
    } else {
      turnLeft();
    }
  }
}
